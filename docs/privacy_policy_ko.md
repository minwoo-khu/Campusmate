# 개인정보처리방침 (초안)

최종 업데이트: 2026-02-12

CampusMate(이하 "앱")는 사용자의 개인정보를 최소한으로 수집하며, 핵심 기능은 로컬 저장(Local-first) 방식으로 동작합니다.

## 1. 수집하는 정보
- 앱 내 사용자가 직접 입력한 데이터
- 할 일, 강의, PDF 메모, 시간표 이미지, 설정값

## 2. 정보의 저장 위치
- 위 데이터는 기본적으로 사용자 기기 내부(로컬)에 저장됩니다.
- 앱은 로컬 알림 기능을 위해 기기 알림 권한을 요청할 수 있습니다.

## 3. 외부 전송
- 앱은 사용자가 입력한 학교 ICS URL을 통해 캘린더 데이터를 조회할 수 있습니다.
- 사용자가 URL을 입력하지 않으면 해당 네트워크 요청은 발생하지 않습니다.
- 그 외 사용자 데이터를 개발자 서버로 업로드하지 않습니다.

## 4. 백업/복원
- 사용자는 설정에서 백업 파일을 생성하고 복원할 수 있습니다.
- 백업 파일 관리(저장/공유/삭제) 책임은 사용자에게 있습니다.

## 5. 제3자 서비스
- 앱은 Flutter 및 안드로이드/iOS 시스템 API를 사용합니다.
- 광고 SDK, 사용자 추적 SDK는 포함하지 않습니다.

## 6. 권한
- 알림 권한: Todo 리마인더 알림 표시
- 파일 접근(사용자 선택 기반): 시간표 이미지/PDF 선택 및 백업 파일 선택

## 7. 아동 개인정보
- 앱은 아동을 별도 타깃으로 하지 않으며, 고의로 아동 개인정보를 수집하지 않습니다.
