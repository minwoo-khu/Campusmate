# CampusMate 개인정보처리방침

시행일: 2026년 2월 13일
최종 수정일: 2026년 2월 13일

CampusMate(이하 "앱")는 이용자의 개인정보를 중요하게 생각하며, 관련 법령을 준수하기 위해 노력합니다. 본 개인정보처리방침은 앱에서 어떤 정보를 어떤 목적으로 처리하는지 설명합니다.

## 1. 개인정보처리자
- 서비스명: CampusMate
- 문의처: minwooj02@gmail.com

## 2. 처리하는 정보 항목
앱은 회원가입 기능이 없으며, 이용자가 직접 입력한 데이터 중심으로 동작합니다.

1) 이용자가 앱에 직접 입력/저장하는 정보(기기 내부 저장)
- 할 일(Todo), 강의명, PDF 관련 메모, 시간표 이미지, 설정값(테마/언어/시작 탭 등)
- 학교 일정 연동을 위해 이용자가 입력한 ICS URL

2) 앱 기능 사용 중 생성되는 기술 정보
- 로컬 알림 기능을 위한 알림 설정/예약 정보
- 백업/복원 기능 사용 시 생성되는 백업 파일

3) 제3자 SDK를 통해 처리될 수 있는 정보
- Google Mobile Ads SDK(AdMob): 광고 식별자, 기기 정보, 앱 진단 정보 등 광고 제공/성과 측정에 필요한 정보
- Sentry(선택적 활성화): 크래시 로그, 스택트레이스, 기기/OS/앱 버전 등 기술적 진단 정보

## 3. 정보 처리 목적
- 할 일/캘린더/시간표/강의자료 관리 기능 제공
- 학교 ICS 일정 조회 및 캘린더 표시
- 로컬 알림 및 리마인더 제공
- 백업/복원 기능 제공
- 광고 제공 및 서비스 운영 안정성/품질 개선

## 4. 정보 처리 방식 및 저장 위치
- 앱의 핵심 사용자 데이터는 기본적으로 이용자 기기 내부(로컬 저장소)에 저장됩니다.
- 이용자가 ICS URL을 입력한 경우에만 해당 URL로 네트워크 요청을 수행합니다.
- 앱은 이용자 콘텐츠를 개발자 운영 서버로 별도 업로드하지 않습니다.

## 5. 보유 및 이용 기간
- 기기 내부에 저장된 정보: 이용자가 삭제하거나 앱을 제거할 때까지
- 백업 파일: 이용자가 저장/공유한 위치에서 이용자가 직접 관리
- 제3자 SDK 처리 정보: 각 제3자 사업자의 정책 및 보관 기간에 따름

## 6. 제3자 제공 및 처리위탁
앱은 자체 백엔드 서버로 이용자 콘텐츠를 수집/판매하지 않습니다. 다만 아래 SDK 제공사가 각자의 정책에 따라 데이터를 처리할 수 있습니다.

1) Google Mobile Ads (AdMob)
- 목적: 광고 제공, 광고 성과 측정, 부정행위 방지
- 관련 정책: https://policies.google.com/privacy
- 추가 안내: https://support.google.com/admob/answer/6128543

2) Sentry (선택적)
- 목적: 앱 오류 분석 및 안정성 모니터링
- 활성화 조건: 앱 빌드 시 `ENABLE_SENTRY=true` 및 유효한 `SENTRY_DSN` 설정 시에만 활성화
- 관련 정책: https://sentry.io/privacy/

## 7. 국외 이전 가능성
AdMob 또는 Sentry 사용 시, 해당 사업자의 인프라 운영 위치(해외 포함)로 정보가 이전되어 처리될 수 있습니다. 세부 사항은 각 사업자의 개인정보처리방침을 따릅니다.

## 8. 이용자의 권리 및 행사 방법
이용자는 개인정보 관련 권리를 행사할 수 있습니다.
- 앱 내 데이터 삭제: 해당 항목 직접 삭제 또는 앱 삭제
- 백업 파일 삭제: 저장 위치에서 이용자가 직접 삭제
- 광고 식별자 관리: 기기 OS 설정에서 재설정/제한 가능
- 문의/요청: minwooj02@gmail.com

회원 계정 기반 서버 저장소가 없으므로, 개발자가 원격에서 개별 이용자 로컬 데이터를 직접 열람/삭제하는 방식은 제공되지 않습니다.

## 9. 개인정보 보호를 위한 조치
앱은 보안 강화를 위해 다음 조치를 적용합니다.
- 백업 PIN 기반 백업 파일 암호화 기능 제공
- 경로/입력값 검증 및 파일 크기 제한 등 안전장치 적용
- 비암호화(HTTP) URL 제한(ICS는 HTTPS만 허용)

## 10. 아동의 개인정보
앱은 아동(만 13세 미만)을 대상으로 설계되지 않았으며, 아동의 개인정보를 고의로 수집하지 않습니다.

## 11. 개인정보처리방침 변경
관련 법령, 기능 변경, 정책 변경이 있는 경우 본 방침을 업데이트할 수 있습니다. 중요한 변경이 있는 경우 앱 업데이트/스토어 공지 등을 통해 안내합니다.
