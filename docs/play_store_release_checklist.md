# Play Store Release Checklist

## 빌드
- [x] `flutter analyze` 통과
- [x] `flutter test` 통과
- [x] `flutter build appbundle --release` 성공
- [ ] 앱 버전(`version`) 증가

## 필수 정책
- [ ] 개인정보처리방침 URL 준비
- [ ] 데이터 보안(Data safety) 설문 작성
- [ ] 콘텐츠 등급 설문 완료
- [ ] 타깃 연령/광고 여부 설정

## 스토어 리스팅
- [ ] 앱 이름/짧은 설명/자세한 설명 작성
- [ ] 연락처 이메일 등록
- [ ] 카테고리 선택
- [ ] 아이콘(512x512) 준비
- [ ] 피처 그래픽(1024x500) 준비
- [ ] 휴대전화 스크린샷 최소 2장 이상

## 캠퍼스메이트 권장 점검
- [ ] 기본 언어 한국어 확인
- [ ] 설정 > 언어 전환(한국어/영어) 동작 확인
- [ ] 설정 > 데이터 백업/복원 동작 확인
- [ ] Todo 리마인더/스누즈 동작 확인
- [ ] 시간표 이미지 업로드/삭제 확인
- [ ] 강의 자료(PDF) 열기 및 메모 확인

## 배포 후
- [ ] 내부 테스트 트랙 배포
- [ ] 크래시/ANR 모니터링
- [ ] 사용자 피드백 반영 이슈 생성
