## 자세한 설명

CampusMate는 대학 생활에서 자주 쓰는 기능을 하나로 모은 로컬 중심 플래너입니다.  
해야 할 일, 일정, 시간표, 강의자료를 앱 하나에서 빠르게 정리할 수 있습니다.

핵심 기능
- 홈: 오늘 요약, 오늘 팁, 학습 현황
- 할 일: 빠른 입력, 마감일/리마인더/반복/우선순위, 필터
- 캘린더: 월간 보기 + Todo/학교 ICS 일정 통합
- 시간표: 이미지 업로드, 확대/축소 확인
- 강의: 강의별 PDF 자료 관리, 전체/페이지 메모, 태그 검색
- 설정: 테마 프리셋, 한국어/영어 전환, 백업 내보내기/복원, 백업 PIN 암호화

데이터/프라이버시
- 대부분의 데이터는 기기에 저장됩니다.
- 네트워크는 필요한 기능에서만 사용됩니다.
  - 사용자가 등록한 학교 ICS URL 조회
  - 광고 SDK 동작(릴리즈 설정에 따름)

3. 캘린더: 월간 일정 + 날짜별 목록
4. 시간표: 이미지 업로드/확대
5. 강의: PDF 자료 + 페이지 메모
6. 설정: 언어/테마/백업 PIN
