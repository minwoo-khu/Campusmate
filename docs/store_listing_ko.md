# Play Store Listing Draft (Korean)

## 앱 이름
CampusMate

## 짧은 설명 (80자 내)
할 일, 캘린더, 시간표, 강의자료를 한 앱에서 관리하는 로컬 우선 캠퍼스 플래너

## 자세한 설명
CampusMate는 대학 생활에 필요한 핵심 기능을 한곳에 모은 로컬 우선(Local-first) 생산성 앱입니다.

주요 기능:
- 할 일(Todo)
- 빠른 입력(Quick Capture)
- 우선순위/반복/리마인더
- 리마인더 스누즈(10분/1시간)
- 캘린더
- 월간 보기 + 학교 ICS 일정 연동
- 시간표
- 이미지 업로드 후 확대/축소 확인
- 강의
- 강의별 PDF 자료 관리
- PDF 메모 및 페이지 메모
- 설정
- 다크/라이트 테마
- 한국어/영어 전환
- 데이터 백업/복원

개인정보/데이터:
- 대부분의 데이터는 기기 내부에 저장됩니다.
- 사용자가 입력한 ICS URL이 있을 때만 해당 일정 조회를 수행합니다.

## 스크린샷 구성안 (휴대전화)
1. Todo 화면: 섹션/필터/스누즈 가능 메뉴
2. Calendar 화면: 월간 + 일정 상세 카드
3. Timetable 화면: 이미지 업로드/확대
4. Courses 화면: 강의 카드 + 자료/메모 통계
5. Settings 화면: 언어/테마/백업복원

## 피처 그래픽 카피 예시
- "캠퍼스 라이프를 한 화면에"
- "Todo · Calendar · Timetable · Courses"

## 제출 전 체크
- 스크린샷 텍스트와 앱 실제 UI 문구 일치
- 한국어 기본, 영어 전환 정상 동작
- 개인정보처리방침 URL 연결 확인
