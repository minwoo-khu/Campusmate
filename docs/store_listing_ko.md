## 자세한 설명

CampusMate는 대학 생활에서 자주 쓰는 기능을 한 앱으로 묶은 로컬 우선(Local-first) 플래너입니다.
할 일, 캘린더, 시간표, 강의자료(PDF/메모)를 한 흐름으로 관리할 수 있습니다.

핵심 기능
- 홈 대시보드: 오늘 요약, 오늘 팁, 학습 현황
- 할 일(Todo): 빠른 입력, 마감일/리마인더/반복/우선순위, 진행/완료 필터
- 캘린더: 월간 보기 + 할 일/학교 ICS 일정 통합 표시
- 시간표: 이미지 업로드, 확대/축소 확인, 강의명 선택 인식 보조
- 강의: 강의별 PDF 자료 관리, PDF 메모/페이지 메모/태그 검색
- 설정: 테마 프리셋, 한국어/영어 전환, 백업 내보내기/복원, 백업 PIN 암호화

데이터 및 프라이버시
- 대부분의 데이터는 기기 내부에 저장됩니다.
- 네트워크는 필요한 기능에서만 사용됩니다.
  - 사용자가 직접 등록한 학교 ICS URL 동기화
  - 광고/진단 도구(릴리즈 설정에 따름)

CampusMate는 복잡한 계정 가입 없이, 학교 생활에 필요한 핵심 기능을 빠르게 쓰는 데 집중합니다.

5. 강의: PDF 자료 + 페이지 메모
6. 설정: 언어/테마/백업 PIN
